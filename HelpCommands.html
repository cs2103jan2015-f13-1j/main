Welcome to Ontask!

Our product will provide the fundamental needs for tracking tasks and events.

Here are the list of commands:

  • Add a floating task by specifying description only (no date and time required).

add <task description>

e.g. add Learn guitar at my own pace.

  • Add a timed tasked event that requires starting date, start time, end date and end time.

add <task description> from <date> <time> to <date> <time>

e.g. add Chalet with friends from 30 Apr 1pm to 1 May 8pm

add <task description> <date> <time> <date> <time>

e.g. add Chalet with friends 30/4 13:00 1/5 20:00

add <task description> <date> <time> <time>

e.g. Meet professor 30 Apr 2015 3pm 5pm 

  • Add a due task that has a due date and time.

add <task description> by <date> <time>

e.g. add complete assignment4 by Friday 23:59

add <task description> <date> <time>

e.g. add complete assignment4 fri 11.59pm

  • To delete a completed task, refer to index number from Task list.

delete <index>

  • To undo most recent action.

undo 

<CTRL+Z key>

  • To redo most recent undo.

redo

  • To edit existing task, state parameter to edit and refer to index number from Task list.

edit task <index> <new task description>

e.g. edit task 1 Meet another professor

edit starttime <index> <time>

e.g. edit starttime 4 3pm

edit endtime <index> <time>

e.g. edit endtime 2 14:00

edit startdate <index> <date>

e.g. edit startdate 1 30 apr

edit enddate <index> <date>

e.g. edit enddate 1 30-3

  • To search for existing task, state parameter to search and the keyword

search task <keyword>

e.g. search task apple

search type (floating, due, deadline ,event, timedtask)

e.g. search type timedtask

search before <date>

e.g. search before 30 Apr

search date <date>

e.g. search date 30-3

  • To return to main task list after search, just press enter key.

<ENTER key>

  • To mark task as completed

mark <index>

  • To flag task to prioritise

flag <index>

prioritise <index>

  • To clear/ reset list to a clean display

clear

  • To exit Ontask application

exit

quit

Time Formats:

Format Examples
24-hour clock 09:00, 13:00, 23:59, 01:25           
12-hour clock 9am, 1pm, 11.59pm, 1.25am

Date Formats:

Format Examples
Day Month Year 3 1 2015, 3-1-2015, 3/1/2015, 3 Jan 2015
Day Month 3 Jan, 3 1, 3-1
Day of week Monday, mon, thursday, thu
Special word today, tdy, tomorrow, tmr
